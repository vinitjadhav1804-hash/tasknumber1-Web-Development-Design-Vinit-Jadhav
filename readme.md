# Task 1: Personal Portfolio Website

## Objective
Create a personal portfolio website to showcase Vinit Jadhav's profile.

## Tech Stack
- HTML5
- CSS3 (Custom Properties, Flexbox, Grid, Animations)
- Vanilla JavaScript

## Sections
- **Header** – Fixed navigation with smooth scroll & active link highlighting
- **Home / Hero** – Name, title, description, CTA buttons, animated avatar
- **About Me** – Cards (Developer, Designer, Innovator), bio, and stats
- **Skills** – Animated progress bars for Frontend and Tools, tech stack icons
- **Contact** – Contact info cards + functional contact form

## Features
- Single-page layout with smooth scrolling
- Active navigation link on scroll
- Skill bars animate on scroll into view
- Responsive design for mobile and desktop
- Dark modern theme with gradient accents

## How to Run
Open `index.html` in any modern browser.
